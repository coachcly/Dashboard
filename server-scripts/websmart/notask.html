Runtime Error

You ran a ProGen WebSmart program without specifying a valid value for the TASK parameter. As a result, the program did not do anything. Please check with the person responsible for constructing this program to determine what a valid value for the TASK parameter is.


If you are the developer, you should review the PML in the definition that produced this program. Usually, TASK=DEFAUL T is valid.
